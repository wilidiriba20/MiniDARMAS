# MiniDARMAS

MiniDARMAS (Mini Digital Audio Recording and Meeting Administration System) is a desktop-based application developed to manage meetings, agendas, recordings, transcriptions, and approvals in an organized and efficient way.

## Project Overview
The system is designed to help institutions digitally manage:
- Meeting scheduling
- Agenda creation
- Audio recording assignments
- Transcription workflow
- Approval and archiving of meeting records

MiniDARMAS reduces manual paperwork and improves transparency and efficiency in meeting documentation.

## Features
- User authentication and role-based access (Admin, Chairperson, Secretary, Transcriber)
- Meeting creation and management
- Agenda and recording management
- Assignment of recordings to transcribers
- Transcription status tracking
- Approval and finalization of meeting records
- Secure database storage

## Technologies Used
- Programming Language: C#
- Framework: .NET (Windows Forms)
- Database: Microsoft SQL Server
- Data Access: ADO.NET
- IDE: Visual Studio

## System Architecture
- Presentation Layer (Windows Forms)
- Business Logic Layer (BLL)
- Data Access Layer (DAL)
- SQL Server Database

## Installation and Setup
1. Clone the repository:
2. Open the project in Visual Studio
3. Restore NuGet packages (if any)
4. Configure the SQL Server connection string
5. Run the database scripts to create tables
6. Build and run the application

## How to Use
1. Login using your assigned role
2. Create and manage meetings
3. Add agendas and recordings
4. Assign recordings for transcription
5. Review and approve transcriptions
6. Finalize and archive meetings

## Academic Purpose
This project was developed for academic purposes to demonstrate:
- Database design
- Layered application architecture
- Transaction handling
- Software engineering principles
